Yet Another Todo Application
Author: Vladimir Fekete

Main actions
 - Task:
    User can add a new task
     [optional] task name = oneliner describing task
     [optional] deadline
     [optional] dependency task (can be mor than one)
     [optional] can be assigned to project (subproject)
     [optional] set importance level
     [optional] severity level
     [optional] description
        - Single click to make mark the task done
        - Move task to another project
        - Delete task
        - Clone task
        - sort (filters: name, projecct, deadline, importance level, severity level)
        - show daily reminders
        - search in tasks (including completed ones)
        - restart task

 - Projects (alternative name: categories)
    - tree hierarchy    
    - tasks can be grouped based on the project

 - view
    - lines with priority vs project tree (+ listbox with priorities) vs tabs as main projects
    - bar graph: tasks and their timespan on time trend

 - GNOME taskbar integration
