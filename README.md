# YATA - Yet Another Todo Application

# What application offers
 - simple todo application for GNOME / Ubuntu (primary platform)
 - gives user possibility to
     - Add task. 
     - Remove task.
     - Cancel task.
     - Mark task as done     
     - view tasks in list
     - view tasks in calendar by the time they started
     - view tasks in calendar by the time they should be done, if any
     - view tasks in timeline graph
     - view only active task
     - view history of finished or canceled tasks
 - Task is a oneliner, no special description or additional
   info.
 - It is expected user knows what the task needs from few words.
